## ğŸ“ Spacing & Sizing

### Padding
- `p-4`: Standard padding (cards, sections)
- `p-6`: Large padding (modals, headers)
- `px-4`: Horizontal padding
- `py-3`: Vertical padding

### Gaps
- `gap-3`: Small gaps between items
- `gap-4`: Standard gaps
- `space-y-4`: Vertical spacing

### Button Sizes
- Height: 48px+ (minimum touch target)
- Padding: `py-3 px-4` (standard button)
- Padding: `py-4 px-4` (large action buttons)

### Card Sizing
- Width: Full (100%, with margins)
- Padding: `p-4` or `p-6`
- Border radius: `rounded-2xl` (16px)
- Shadows: `shadow-sm` to `shadow-md`

### Hover States
- Buttons: Slightly darker color
- Cards: Shadow increases
- Images: Scale up 105%

- Buttons: Scale down 95%
- Tabs: Lime-600 background
