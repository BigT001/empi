## Using the Features

### View Dashboard Overview
1. Go to `/dashboard`
2. Dashboard tab is active by default
3. See all your profile info and stats
4. View recent orders

### View All Invoices
1. Click the "Invoices" tab
2. See all your invoices as professional receipts
3. Each invoice shows complete details

### Print an Invoice
1. Go to Invoices tab
2. Click the blue "Print Receipt" button
3. Print dialog opens
4. Choose printer and settings
5. Click Print

### Download an Invoice
1. Go to Invoices tab
2. Click the purple "Download" button
3. File downloads as `Invoice-[Number].html`
4. Save to your computer

## Troubleshooting

### Dashboard not loading?
1. Try refreshing page
2. Check internet connection
3. Try different browser
4. Clear browser cache

### Invoices not showing?
1. Verify you have placed orders
2. Check payment was completed
3. Refresh page
4. Try different browser

### Print dialog not opening?
1. Check popup blockers
2. Allow popups for this site
3. Try different browser
4. Try downloading instead

### Download not working?
1. Check downloads folder
2. Check browser download settings
3. Try different format (print to PDF)
4. Try different browser
